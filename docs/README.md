# 기능목록

## 다리 만들기

- 다리의 길이를 입력 받는다
- 길이 만큼 0과 1을 무작위로 생성하여 0이면 D, 1이면 U를 더한다.

## 다리 건너기

- 건널 수 있는 칸이 위일 때 U를 입력받으면 O표시
- 건널 수 있는 칸이 위일 때 D를 입력받으면 X표시
- 건널 수 있는 칸이 아래일 때 D를 입력받으면 O표시
- 건널 수 있는 칸이 아래일 때 U를 입력받으면 X표시

## 예외 처리 및 게임 재시작

- 입력 예외가 발생하거나 잘못된 칸으로 이동한 경우 게임을 재시작할 수 있따.
- 게임 시도 횟수를 센다
- 재시작 시 기존의 다리를 재사용한다.
